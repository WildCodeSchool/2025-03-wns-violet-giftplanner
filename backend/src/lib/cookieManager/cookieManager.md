# 🍪CookiManeger  

Mini framework de gestion de cookies

## ✨ Objectif
- Avoir un accès facile aux cookies de nimporte ou dans le projets

## 🚀 Installation  
Copier les fichiers du mini framework dans un dossier `lib/cookiManeger/`  

**Arborescence du projet :**

```
projet/
└── lib/
    └── cookieManager/
        ├── cookieManager.md
        ├── cookieManager.ts
        ├── types/
        │   └── cookieLibType.ts
        └── utils/
            ├── delCookie.ts
            ├── getCookie.ts
            └── setCookie.ts
```
